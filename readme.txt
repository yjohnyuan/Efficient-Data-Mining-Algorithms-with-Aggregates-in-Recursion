
To run datalog program, just run:
chmod +x datalog-run.sh
./datalog-run.sh

To run RaSQL program, just run:
chmod +x rasql-run.sh
./rasql-run.sh

Note:
To guanrantee the program can run successfully, the sudo keywords are included in the two scripts. If it can works well without using sudo on your computer, feel free to remove "sudo" from above two scripts.
 
No argument is required in these two scripts. To change the set of queries and the dataset for each query, you need to change the files under "query" and "testdata" directory. More details are shown in the tutorial slides for the system.
